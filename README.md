# Open Source Audit — Git

Hi, I’m Anant Kumar (24BCE11278).
This is my project for Open Source Software.

I chose Git because its story is interesting. When the Linux team lost access to their old tool, Linus Torvalds made Git in about 10 days. That idea — building your own tool instead of depending on others — is what I liked the most.

In this project, I tried to understand Git beyond just commands. I studied where it came from, how its license (GPL v2) works, and why open source matters. I also looked at how Git works on a Linux system and how it is used in real development.

I also made 5 small bash scripts:

* One shows system details like user, kernel, and uptime
* One checks if a package is installed
* One looks at disk usage and permissions
* One reads log files and finds errors
* One creates a simple open source manifesto

These scripts helped me understand how Linux actually works.

From this project, I learned that Git is not just a tool. It changed how people work together. Most modern development depends on it, even if we don’t notice it.
